## 빌드 하는 법

1. Repository를 clone!
2. project root로 가서 `./gradlew build` 실행
3. 빌드파일로 이동
    * `cd <proj-root>/build/libs/`
4. 빌드파일 실행
    * `java -jar leadpet-0.0.1-SNAPSHOT.jar`

## 프로젝트 실행 하는 법

1. Repository를 clone!
2. project root로 가서 `./gradlew build` 실행
3. 프로젝트 실행
    * `./gradlew bootrun`
